## How to Use These Resources

**For Learning VID:**
1. Start with the main book chapters in `/chapters`
2. Use **VID-One-Page-Overview.md** as a quick reference
3. Review **VID-Real-World-Examples.md** for context

**For Teaching VID:**
1. Use **VID-Diagrams.md** for presentations
2. Share **VID-For-Engineering-Managers.md** with leadership
3. Provide **VID-Templates.md** for hands-on practice

**For Implementing VID:**
1. Reference **VID-Templates.md** for starting points
2. Adapt examples from **VID-Real-World-Examples.md**
3. Use metrics from **VID-For-Engineering-Managers.md**

## Relationship to Main Book

These resources supplement the core methodology described in the `/chapters` folder:

- **Main Book** - Complete methodology, principles, and practices
- **Resources** - Quick references, templates, and specialized guides

For the full VID methodology, read the book chapters. Use these resources for quick reference, teaching, or specialized needs.
