# VID Resources

This directory contains supplementary resources for Verified Intent Development.

## Available Resources

### Visual Diagrams
- **[VID-Diagrams.md](VID-Diagrams.md)** - Visual representations of VID concepts
  - Workflow visualizations
  - Principle illustrations
  - Decision trees
  - Use in presentations and teaching

### Practical Templates
- **[VID-Templates.md](VID-Templates.md)** - Ready-to-use templates
  - Intent specification templates
  - Verification checklists
  - Code review templates
  - Documentation formats
  - Git commit templates
  - PR templates

## How to Use These Resources

**VID-Diagrams.md:**
- Include in presentations about VID
- Use to explain concepts visually
- Reference when teaching the methodology
- Embed in documentation

**VID-Templates.md:**
- Copy and customize for your projects
- Use as starting points for your own templates
- Adapt to your team's workflow
- Integrate into your development tools

## Integration with VID Book

These resources are referenced throughout the VID methodology book and complement the core content in the `/chapters` folder.
