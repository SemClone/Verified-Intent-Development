## How to Add Your Name

If you've contributed to VID, please add your name to this file:

1. Fork the repository
2. Add your name in the appropriate section (alphabetically by last name)
3. Include a brief description of your contribution
4. Submit a pull request

### Contribution Guidelines

You can add your name to the appropriate section if you have:

**Core Contributors:**
- Authored or significantly revised major sections of the methodology
- Developed substantial tools or frameworks for VID
- Provided significant real-world case studies or research
- Made sustained contributions over time

**Contributors:**
- Fixed documentation errors or improved clarity
- Added examples or use cases
- Improved tooling or scripts
- Translated content
- Any other valuable contribution
