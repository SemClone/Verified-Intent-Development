# VS Code Snippets Installation

## Quick Install

1. **Open VS Code**
2. **Press**: `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
3. **Type**: "Snippets: Configure User Snippets"
4. **Select**: The language you want (e.g., "javascript", "typescript", "python", etc.)
5. **Paste** the contents of `provenance-markers.code-snippets` into the file
6. **Save**

## Usage

After installation, you can use these snippets by typing the prefix and pressing Tab:

| Prefix | Description | Usage |
|--------|-------------|-------|
| `vid-ai` | AI-generated code marker | Use when code was fully generated by AI |
| `vid-assist` | AI-assisted code marker | Use when you wrote code with AI suggestions |
| `vid-human` | Human-original code marker | Use when code is entirely human-written |
| `vid-mixed` | Mixed provenance marker | Use when code combines human and AI work |
| `vid-verify` | Verification notes | Add verification details to existing code |

## Example Usage

1. Type `vid-ai` in your code editor
2. Press `Tab`
3. Fill in the placeholders:
   - Tool name (e.g., "copilot", "chatgpt")
   - Verification level (deep/standard/light/unreviewed)
   - Your email
   - Human intent description
   - Known limitations

Result:
```typescript
// @provenance ai-generated
// @tool copilot
// @verification deep
// @reviewer alice@example.com
// Human intent: Implement OAuth token refresh with retry logic
// Known limitations: Doesn't handle network partitions > 30s
function refreshToken(token: string) {
  // your code here
}
```

## Language-Specific Installation

The snippet format shown uses `//` for comments (JavaScript/TypeScript style). For other languages, modify the comment syntax:

### Python
Replace `//` with `#`:
```python
# @provenance ai-generated
# @tool copilot
# @verification deep
```

### Ruby
Replace `//` with `#`:
```ruby
# @provenance ai-generated
# @tool copilot
# @verification deep
```

### HTML/XML
Replace `//` with `<!--` and `-->`:
```html
<!-- @provenance ai-generated -->
<!-- @tool copilot -->
<!-- @verification deep -->
```

## Global Snippets (All Languages)

To make snippets available in all file types:

1. Press `Cmd+Shift+P` / `Ctrl+Shift+P`
2. Type "Snippets: Configure User Snippets"
3. Select "New Global Snippets file..."
4. Name it "vid-provenance"
5. Paste the contents of `provenance-markers.code-snippets`

Now the snippets work in any file!

## Alternative: Workspace Snippets

To share snippets with your team:

1. Create `.vscode/vid-provenance.code-snippets` in your project root
2. Paste the snippet contents
3. Commit to version control
4. Team members will automatically get the snippets when they open the workspace
